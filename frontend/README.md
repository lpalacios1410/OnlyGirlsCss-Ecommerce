# Frontend (React + Vite)

Este directorio contiene la aplicación de frontend construida con React y Vite.

## Scripts principales

- `pnpm dev` / `npm run dev`: Inicia el servidor de desarrollo.
- `pnpm build` / `npm run build`: Genera la build de producción.
- `pnpm preview` / `npm run preview`: Previsualiza la build de producción.

## Instalación

```bash
pnpm install # o npm install
```

## Notas

- El archivo `vite.config.js` y las configuraciones de ESLint están adaptadas para esta estructura.
- Los assets y recursos públicos están en `public/`.
