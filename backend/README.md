# Backend (Express)

Este directorio contiene la API y lógica de servidor usando Express.

## Scripts principales

- `npm install`: Instala las dependencias.
- `node app.js`: Inicia el servidor backend.

## Notas

- El archivo principal es `app.js`.
- Las rutas, middlewares, modelos y controladores están organizados en subcarpetas.
- El archivo `package.json` está adaptado para esta estructura.
